```

$ git clone git@github.com:Miguel-Oliveiraa/react-native-food-app.git
```


## ☕ Using <rocket-help>

To use <food-app>, you must:
Enter the directory
```
$ cd react-native-food-app
```
install dependencies
```
$ npm install
or
$ yarn
```
Run the expo app
```
$ npx expo start
```
